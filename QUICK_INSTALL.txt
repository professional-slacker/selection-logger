=== Selection Logger クイックインストール ===

1. selection-logger.exe をダウンロード

2. 管理者としてコマンドプロンプトを開く：
   - Windowsキー + X
   - 「Windows PowerShell (管理者)」または「コマンド プロンプト (管理者)」

3. 以下のコマンドを実行：

   mkdir "C:\Program Files\SelectionLogger"
   copy selection-logger.exe "C:\Program Files\SelectionLogger\"
   cd "C:\Program Files\SelectionLogger"
   selection-logger.exe install
   selection-logger.exe start

4. 警告が表示された場合：
   - ファイルを右クリック →「プロパティ」
   - 「全般」タブ →「許可する」にチェック
   - 「OK」をクリック

=== アンインストール ===

   cd "C:\Program Files\SelectionLogger"
   selection-logger.exe stop
   selection-logger.exe uninstall
   cd \
   rmdir /s "C:\Program Files\SelectionLogger"

=== ログの場所 ===
Documents\SelectionLogs\  （例：2025-04.txt）

=== コマンド ===
selection-logger.exe status   # 状態確認
selection-logger.exe stop     # 停止
selection-logger.exe start    # 開始
selection-logger.exe --help   # ヘルプ
